## <u>Teamate</u> Project Background
- ### 필요성
기존 모바일 FPS 게임은 현실감이 부족하며, 플레이어의 몰입도를 높이는 데 한계가 있음

증강현실(AR) 기술의 대중화로 실제 공간과 가상 전투를 결합한 새로운 형태의 게임에 대한 수요 증가

운동, 경쟁, 소셜 요소를 결합한 체험형 엔터테인먼트에 대한 관심 확산
- ### 기존 해결책의 문제점
단순한 카메라 필터 수준의 AR 구현으로 실제 전투 몰입감이 낮음

GPS 기반 AR 게임(예: Pokémon GO)은 ‘탐험’ 중심으로, 전투형 상호작용이 부족함

대부분의 AR 게임은 FPS의 핵심 요소인 조준, 반응속도, 전투 전략성을 충분히 반영하지 못함

## System Design
  - ### System Requirements
AR Tracking: 스마트폰 카메라를 이용한 실시간 환경 인식 및 3D 몬스터 배치

Weapon Interaction: 화면 터치 및 자이로센서를 이용한 조준·사격 시스템

Game Engine: Unity 기반, AR Foundation 및 ARCore/ARKit 지원

Performance Optimization: 저사양 기기에서도 안정적인 FPS 유지 (60fps 목표)

Surround Sound : 서라운드 기능을 추가해 현실감 극대화

## Case Study
  - ### Description
플레이어는 현실 공간(예: 집, 공원, 거리)에 나타나는 가상의 몬스터를 스마트폰을 통해 탐지하고, 직접 사격하여 처치

몬스터는 랜덤하게 등장하며, 이동 및 공격 AI를 탑재

거대 보스 공략 가능

향후 확장 방향: PvP 모드, 지역 기반 이벤트, 웨어러블 기기 연동
  
  
## Conclusion
  - 본 프로젝트는 증강현실과 FPS 장르의 융합을 통해 현실 공간을 새로운 게임 전장으로 확장한 시도입니다.
향후 PvP 모드, 지역 기반 이벤트, 웨어러블 기기 연동 등을 통해 지속적 확장성과 상용화 가능성을 갖춘 콘텐츠로 발전할 것입니다.
